# dotfiles

harysys's dotfiles managed by [chezmoi](https://www.chezmoi.io/).

## Get started

1. Install brew

```sh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

2. Install chezmoi

```sh
brew install chezmoi
```

3. Initialize dotfiles (clone & set config)

```sh
chezmoi init harusys
```

4. Apply dotfiles

```sh
chezmoi apply
```

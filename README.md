# dotfiles

harysys's dotfiles managed by [chezmoi](https://www.chezmoi.io/).

## Get started

1. Install chezmoi
```sh
sh -c "$(curl -fsLS get.chezmoi.io)"
```

2. Clone dotfiles
```sh
chezmoi init harusys
```

3. Set chezmoi configurations

```sh
chezmoi edit-config
```

4. Apply dotfiles
```sh
chezmoi apply
```
